Update operators updateOne() updateMany()

These methods let you change existing documents by matching a filter and applying update operations like $set, $inc, etc.

üîß 1. updateOne()

Updates the first document that matches the filter.

üìò Syntax:

db.collection.updateOne(filter, update, options)

üîπ Example:

db.users.updateOne(
  { name: "Alena" },
  { $set: { age: 31 } }
)

üî∏ This will find the first user named Alena and set age to 31.

üîß 2. updateMany()

Updates all documents that match the filter.

üìò Syntax:

db.collection.updateMany(filter, update, options)

üîπ Example:

db.users.updateMany(
  { isStudent: true },
  { $inc: { age: 1 } }
)

üî∏ This will increase age by 1 for all users where isStudent is true.

‚úÖ Example: Update and Add Field

db.posts.updateOne(
  { postId: 3511 },
  {
    $set: { status: "published" },
    $inc: { comments: 1 }
  }
)

This will:
- Add or update the status field.
- Increment the comments count by 1

üîç Want to see what was updated?

Use updateOne() with option { returnDocument: "after" } in a findOneAndUpdate() operation instead: 

db.users.findOneAndUpdate(
  { name: "Alena" },
  { $set: { age: 32 } },
  { returnDocument: "after" }
)

Operator Description Example
$set Set the value of a field { $set: { age: 30 } }
$unset Remove a field { $unset: { nickname: "" } }
